shysta的杂谈

寒假干了什么呢

寒假想了不少关于未来规划的事，不能说没用，但现在回头看，还是有很多局限。那段时间其实挺矛盾的——嘴上说着想学东西，实际却很难沉下心（其实现在也是）。美其名曰“思考未来”，更像是在给自己找借口不行动。

不过好歹也做了一点事：比如从零搭出了这个博客。原谅我大二才具备这个能力吧，我确实太菜了。

之前在干什么

大一和大二上，我在干什么呢？想想看，每天也就是上课下课，为作业发愁——大一为 C++ 代码发愁，考试前刷题求个好成绩。但最终，到现在我还是个烂 GPA，哈哈，悲。

之前一直幻想着，只要按部就班上课下课，就能成为想象中的 CS er：上个课，读个研，弹着吉他，剪剪视频，很多事情好像都很容易成功。

但可惜事实并非如此——shysta

大二寒假暂且学了深度学习的一点内容，但还是很浅，pytorch基础也没打牢（现在好点了 话说我像跑深度学习也是人生规划的一部分），试图直接跑代码，确实是正确滴。至少寒假的行为给我带来了现在学习的大部分动力。

所以改变这个东西，不要很轻易，过于容易改变被说服反而很容易失去一些东西hiahia

反正已经不会更坏了，不如就安之若素。

开学以来想学的东西很多，比如操作系统前面的计算机基础要补一补，其实本来是看nju的jyy老师操作系统课操作vim的身姿很帅，才决定想要学vim和操作系统之类的。后面又配置了astrbot 就是qq机器人，原本是像找个ai项目，但发现很多东西没有那么容易就直接跑的，GitHub项目大都是很久的沉淀才发出来，什么都不会的我怎么能随意沾染（笑）

所以还是跑了一个qq机器人 tafy 哈哈

  • tafy的qq：3984867181

现在的学习

现在还在学的东西

  • fastapi初步学习
  • [] 前端react初步学习
  • [] vim 学习（顺便补汇编知识）
  • [] 操作系统实验+计基实验
  • [] 深度学习长期
  • [] 探索ing

然后感觉深度学习还是要长期学，后面有机会还是要多写点学习记录。

然后就是有了一个做个GitHub小项目的想法，想接触全栈开发（是这样说的吧）带着想法开始ai对话，找找找，最后选择了fastapi的后端，反正也是python的，至少不亏hiahia。

fastapi学的就是疯狂看文档，连看好几天，算是入了个小门，后面websocket什么的需要的时候再学。

然后准备写前端页面的时候才发现 原来更难的还在前端。又是自不量力尝试直接用架构（跟之前一样，啥子都是幻想不学内容直接用ai写代码然后脑补我怎么这么厉害），被打回原形才开始继续ai对话，找到大致学习路线又开始看文档，刷b站的快速的课。但根基根本不牢啊。但也没办法了。

愿与各位共勉：花有重开日，人无再少年。但也不必太在意得失，抛开一切不谈，你都是一个很好的人。

就这样，新学期慢慢来吧。
